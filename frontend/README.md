# Frontend - Plantilla para proyecto de ingenieria de software

Plantilla de frontend desarrollada para el proyecto final de ingenieria de software.

## Tecnologias

- React 18
- Vite
- React Router DOM
- Axios
- SweetAlert2
- JWT Authentication

## Instalacion

1. Instalar dependencias:
```bash
npm install
```


## Ejecucion

Modo desarrollo:
```bash
npm run dev
```

Compilar para producción:
```bash
npm run build
```

Vista previa de producción:
```bash
npm run preview
```

## Estructura del Proyecto

```
src/
├── assets/         # Imágenes, iconos, etc.
├── components/     # Componentes reutilizables
├── context/        # Contextos de React
├── helpers/        # Funciones auxiliares
├── hooks/          # Hooks personalizados
├── pages/          # Páginas de la aplicación
├── services/       # Servicios de API
└── styles/         # Archivos CSS
```
